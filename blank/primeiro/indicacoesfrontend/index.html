INDICAÇÕES TÉCNICO

W3schools

Um dos sites relacionados à linguagem de programação mais conhecidos é o W3schools. O site oferece uma extensa quantidade de informações relacionadas à HTML, CSS, JAVASCRIPT, PYTHON, e derivados, gratuitamente. Permite, ainda, que o usuário teste os adquiridos, dentro do próprio W3schools.

Mdn_

O segundo site é o Mdn_, sendo o website oficial de Mozilla para desenvolvimento de padrões web e projetos de Mozilla. Assim como o W3schools, o site hospeda diversos documentos sobre uma grande variedade de conteúdos referentes à programação e desenvolvimento web.

Hostinger Tutorials

O Hostinger Tutoriais é um site que descreve detalhadamente o que é programação, desenvolvimento web e suas principais linguagens, além de apresentar uma série de cursos gratuitos disponíveis.

Fundamentos de HTML5 e CSS3

O livro Fundamentos do HTML5 e CSS3 fornece aos iniciantes conceitos básicos de HTML e CSS para iniciar na área da programação e desenvolvimento web.

Fala Dev

"A gente passa a maior parte do tempo escrevendo código. Agora chegou o momento de falar sobre isso. Nesse podcast três programadores se reúnem para discutir o universo da programação."
Fala Dev é um podcast produzido por devs que, de uma maneira leve e divertida, compartilham suas experiências e dicas relacionadas à vida de desenvolvedor web.
