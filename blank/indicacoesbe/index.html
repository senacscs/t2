Tutorial

O Roadmap.sh é um ótimo recurso - online e gratuito - que auxilia os desenvolvedores a escolherem um caminho e guiarem seus aprendizados. Com uma vasta gama de mapas, guias e outros conteúdos sobre diversas ferramentas e tecnologias de programação e desenvolimento web.

A trilha Desenvolvedor Backend é um bom começo para quem deseja iniciar no desenvolvimento backend. "Guia passo a passo para se tornar um desenvolvedor backend moderno."

O guia possui uma comunidade ativa no GitHub, onde é possível interagir com outros desenvolvedores e contribuir com o projeto.

Site

O site Free Code Camp é outro recurso online e gratuito, excelente para aprender a programar e desenvolver aplicações web. Ele oferece cursos interativos com prática dos conhecimentos através de projetos e emissão de certificados. Há cursos tanto de desenvolvimento front-end, quanto de desenvolvimento back-end - como o de Desenvolvimento de Back End e APIs.

É possível escolher os tópicos de maior interesse, mas recomenda-se seguir o currículo proposto pelo site.

Vídeo

"O que é um desenvolvedor backend e o que ele faz" e "Por que se tornar um desenvolvedor backend" são alguns dos tópicos abordados no vídeo "Como se tornar um desenvolvedor BACKEND?" da Attekita Dev.

O vídeo ensina os conceitos básicos de programação para a web, usando diferentes linguagens e frameworks do lado do servidor. Ainda oferece alguns recursos e dicas para quem quer aprender mais sobre o desenvolvimento backend, como tutoriais, cursos, livros e podcasts.

Podcast

O episódio “Entendendo backend para iniciantes” da série “Começando aos 40” do Akita explica os conceitos básicos de programação para a web, usando diferentes linguagens e frameworks do lado do servidor.

O episódio não é uma aula nem um tutorial, mas sim uma forma de conectar os pontos de diversos assuntos que podem ser encontrados de forma desconexa na internet.

Indicações Extra

Além dessas recomendações, também trago duas dicas que combinam um pouco de cada coisa já citada.

Artigo

O artigo “Como se Tornar um Desenvolvedor Backend: Habilidades Importantes para Aprender” do blog site Kinsta é um guia que explica o que é o desenvolvimento backend, quais são as principais habilidades e linguagens de programação necessárias para se tornar um desenvolvedor backend, e como começar nessa carreira.

É um artigo destinado a iniciantes ou pessoas interessadas em se tornar desenvolvedores backend.

Repositório GitHub

Existem muitos vídeos sobre programação e criação de aplicações na internet. Porém, nem sempre é fácil achar conteúdos de qualidade que sejam acessíveis e em português. Por isso, eu recomendo o repositório do GitHub “Canais Brasileiros para Pessoas Desenvolvedoras” que reúne vários canais sobre esse assunto.

O repositório é dividido em várias categorias, como Banco de Dados, Desenvolvimento Back-End, Desenvolvimento Front-End, Desenvolvimento Mobile, Inteligência Artificial, Games, Lógica de Programação e Segurança.
