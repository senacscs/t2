Tutorial

O site Free Code Camp é um bom recurso - online e gratuito - para aprender a programar e desenvolver aplicações web. Ele oferece cursos interativos com prática dos conhecimentos através de projetos e emissão de certificados. Há cursos tanto de desenvolvimento front-end, quanto de desenvolvimento back-end - como o de Desenvolvimento de Back End e APIs.
É possível escolher os tópicos de maior interesse, mas recomenda-se seguir o currículo proposto pelo site.

Site

O Roadmap.sh é outro recurso online e gratuito que auxilia os desenvolvedores a escolherem um caminho e guiarem seus aprendizados. Com uma vasta gama de mapas, guias e outros conteúdos sobre diversas ferramentas e tecnologias de programação e desenvolimento web.
A trilha Desenvolvedor Backend é um bom começo para quem deseja iniciar no desenvolvimento backend. "Guia passo a passo para se tornar um desenvolvedor backend moderno."
O site possui uma comunidade ativa no GitHub, onde é possível interagir com outros desenvolvedores e contribuir com o projeto.
