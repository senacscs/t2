Introdução

A medicina veterinária tem evoluído, explorando alternativas naturais como fitoterápicos para melhorar o bem-estar de cães e gatos. Este projeto aborda os benefícios, cuidados e riscos do uso de medicamentos naturais.

Tema

O trabalho foca no uso de medicamentos naturais na medicina veterinária, conectando saúde, biologia e ciências sociais para melhorar o bem-estar animal.

Problema

Identificar medicamentos naturais seguros e eficazes para cães e gatos.

Objetivos

Objetivo Geral

Orientar tutores sobre medicamentos naturais para animais domésticos.

Objetivos Específicos

  • Pesquisar o uso de medicamentos naturais na veterinária.
  • Entrevistar veterinários sobre práticas relacionadas.
  • Desenvolver um site informativo para tutores.

Metodologia

Foi realizada uma pesquisa em artigos acadêmicos e entrevistas com veterinários para levantar informações sobre o uso de plantas medicinais em tratamentos para animais.

Conclusão

O projeto busca promover práticas sustentáveis e seguras na medicina veterinária, utilizando medicamentos naturais para melhorar a qualidade de vida dos animais.
