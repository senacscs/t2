Tutorial

css(significa Cascading Style Sheets) é um simples mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web. CSS tem uma sintaxe simples, e utiliza uma série de palavras em inglês para especificar os nomes de diferentes estilos de propriedade de uma página.

link para saber mais sobre:https://terminalroot.com.br/


Livro

html and css: Design and Build Websites Todos os dias, mais e mais pessoas querem aprender HTML e CSS. Juntando-se aos web designers e programadores profissionais estão novos públicos que precisam conhecer um pouco de código no trabalho (atualizar um sistema de gerenciamento de conteúdo ou loja de e-commerce) e aqueles que querem tornar seus blogs pessoais mais atraentes. Muitos livros que ensinam HTML e CSS são secos e escritos apenas para aqueles que querem se tornar programadores, e é por isso que este livro tem uma abordagem totalmente nova.
