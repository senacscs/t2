SITE

um site sobre programação pode conter tutoriais, informações, curiosidades e dicas etc... um site pode explicar o que e html entre outras coisas. Tá, mas o que é html e css? Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos Você sabe o que é CSS? Trata-se de uma linguagem de marcação, amplamente utilizada com HTML ou XHTML, e representa diversas possibilidades para a formatação. O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página. E tem muitas outras coisas por isso deixarei a fonte aqui
