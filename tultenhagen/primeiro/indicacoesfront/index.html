O HTML é uma linguagem de marcação, porém para facilitar a compreensão dos leigos, geralmente nos referimos como linguagem de programação. Seu nome deriva da expressão inglesa "Hyper Text Markup Language". Usada comumente na construção base de sites para estruturar seu conteúdo.

Surgiu da combinação dos padrões HyTime e SGML, linguagens complemetares que atualmente podem ter toda ou maior parte de sua função resumida a uma só aplicação.

 Tutorial 

O site w3schools.com é uma fonte incrível para aqueles que querem tutoriais de como fazer uso correto das tags para uma determinada função. É claro em suas didáticas, e nos permite testar as tags em sua própria plataforma, assim conseguimos compreender com maior dimensão o que cada símbolo e nomeclatura indicam dentro de um website.
