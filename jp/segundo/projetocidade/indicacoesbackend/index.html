O que é Back-end

As linguagens de programação backend lidam com os bastidores das funcionalidades das aplicações web. É o código que conecta a internet com o banco de dados, gerencia as conexões dos usuários e alimenta a aplicação web. O backend trabalha em conjunto com o frontend para entregar o produto para o usuário final. Um bom site para você dar uma olhada melhor se encontra aqui.
