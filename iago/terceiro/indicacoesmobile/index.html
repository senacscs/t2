Swift - linguagem

Escolhi falar sobre o Swift por ser uma linguagem de programação popular por ser fácil de aprender e usar, combinando elementos de outras linguagens como o Objective-C, C#, Rust e Python. Sendo uma linguagem usada principalmente para o desenvolvimento de aplicativos e software para dispositivos Apple.

Mas um problema fica no quesito que só se da para programar com o sistema operacional macOS, mas com com a utilização de simuladores ou de emuladores é possível de programar em um computador Windows ou Lynux.
