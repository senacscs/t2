MUSICDOCTOR

O que é esse projeto?

Bem-vindo ao MusicDoctor, uma plataforma que serve de auxílio para as sessões de musicoterapia de forma a trazer perguntas para a pessoa que deseja fazer marcar uma sessão de musicoterapia através de um formulário online, com algumas perguntas como, se já realizou algum outro tratamento como terapias, qual é o problema observado que o paciente deseja resolver, entre outras questões que auxiliarão na organização da sessão para o paciente.

Esta plataforma tem como objetivo também trazer uma explicação sobre a musicoterapia e aproximar o usuário com o musicoterapeuta!

Espero que eu seja capaz auxiliá-lo!
