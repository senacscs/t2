livro

Learn Python the Hard Way

Este livro pode servir como um bom ponto de partida para começar a aprender Python. Ele explica o básico da linguagem e não requer nenhuma experiência prévia em programação para ser compreendido.

Embora o título do livro dê a entender que será muito difícil lê-lo e compreendê-lo, o que é mostrado com o livro é que a maneira mais difícil é a melhor forma de programar, obter bons resultados e se familiarizar com a linguagem de programação.

HTML Tutorial - W3Schools

Neste tutorial, você pode aprender HTML do básico ao avançado. São mostradas desde as tags iniciais de um site, como html, head e body, até a incorporação de mídia e ligação com CSS e Java Script.

podcast

Podcast Fala Dev: acessibilidade no ecossistema de programação

Neste episódio do podcast Fala Dev, os participantes discutem sobre a acessibilidade no desenvolvimento web e levantam reflexões sobre o estado em que ela se encontra. Como tornar a web mais acessível para desenvolvedores e usuários?

Também é relatada a trajetória de pessoas com deficiência que programam e trabalham com TI.
