Estudante do Ensino Médio Técnico

O ensino médio técnico une a formação geral do ensino médio com uma formação técnica específica em uma área profissional, com o objetivo de capacitar os estudantes para o mercado de trabalho.

Nas aulas de programação vemos as linguagens mais utilizadas pelo mercado de trabalho. Aprendemos desde a base, e avançamos ao decorrer dos anos. Alguns exemplos são:


  • HTML
  • CSS
  • JS
  • PHP
  • Flutter
